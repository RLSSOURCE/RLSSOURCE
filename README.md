## 🔥 Featured Project

### [The Licensing Labyrinth — Power BI Dashboard](https://app.powerbi.com/view?r=eyJrIjoiNTFmOGRmZDUtNTg0NC00MGNmLTg5N2EtN2ExYmQzNTAxMjQwIiwidCI6IjIyNTYxMTY5LWFjOTctNGIyZC1hMGExLTU0OTJlNjM5OTA5ZSJ9)

> *A data-driven case study on Microsoft licensing complexity — built out of pure frustration.*

- 📊 Authored **10+ DAX measures** — COUNTROWS-based counts, DIVIDE() rate calculations, and CALCULATE() with FILTER() for the MCA-blocked rate (83.33%) and average frustration score
- 🔍 Modeled licensing-attempt data to surface time lost by category, failure reasons, escalations, and a step-order frustration trend
- 📈 Built an interactive multi-visual report (bar, line, and donut charts) with slicers for category, result, billing account, and support status — published live to the web
